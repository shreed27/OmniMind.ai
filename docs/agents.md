# Agents

## Executive Board

| Agent | Role |
|------|------|
| CEO | final decision, delegation, mission alignment |
| CTO | architecture, implementation, infrastructure |
| COO | execution rhythm, scheduling, blockers |
| CFO | budget, cost tracking, ROI |
| CMO | positioning, messaging, stakeholders |
| Legal | compliance, risk, policy |
| Research | evidence gathering, benchmarking |
| Security | threat modeling, approvals, red team |

## Workers and specialists

Start with minimal roles required by the mission.
Spawn temporary specialists when evidence, compliance, or domain depth exceeds current team capability.

## Agent contract

Each agent owns:
- memory slice
- skill list
- tool/API access
- KPIs
- artifacts
- decisions

Each agent emits events, never direct calls to other agents.
