## ⚡ ULTRA-HIGH VALUE "WOW" FEATURES

### 🧠 WOW FEATURE #1: THE DYNAMIC EXECUTIVE BOARD DEBATE (Powered by Gemini 1.5 Flash)
*   No more black-box prompting. When a user issues an objective, `EnterpriseOS` spins up a live **A2A (Agent-to-Agent) boardroom debate**.
*   **The Conflict**: The **CTO** fights for maximum quality (Frontier LLM tokens), while the **CFO** monitors strict budget limit policies.
*   **The Vote**: If an operational or budget threshold is violated (e.g., spending over $10,000 limit), the system initiates a structured consensus vote. The debate transcript, voter stances, and outcomes are dynamically generated by Gemini and written to the immutable ledger.
*   *Endpoint: `GET /api/v1/executive-board/missions/{id}/debate`*

### 🛠️ WOW FEATURE #2: AI GIT HISTORY COPILOT & PR WRITER
*   An intelligent system-level utility that interfaces directly with local repositories.
*   It analyzes active file diffs (`git diff`) and automatically generates **Conventional Commit messages** and beautiful, structured **Markdown PR descriptions** (including key changes, file modifications, and automated verification checklists).
*   *Endpoint: `POST /api/v1/git/history`*

### 🎨 WOW FEATURE #3: SOCIAL CAMPAIGN & IMAGEN 3 MARKETING COPILOT
*   Converts any raw product description into a complete social media growth campaign.
*   Integrates with **Unsplash guided templates** to fetch theme-specific image assets while **Gemini** writes customized platform-specific copy:
    *   **X/Twitter**: High-impact hooks, max 280-chars, and custom hashtags.
    *   **LinkedIn**: Structured bullet-points, value-driven, and clear CTAs.
    *   **Instagram**: Aesthetic trend copywriting with visual layout formatting.
*   *Endpoint: `POST /api/v1/creative/product-campaign`*

## 🕹️ 1-CLICK HACKATHON RUNBOOK

We have packaged the entire project into a **powerful, 1-Click Interactive CLI Utility** to demo live to judges.

### Step 1: Initialize Environment
Clone the repository and set up your virtual environment:
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r backend/requirements-dev.txt
```

### Step 2: Inject Gemini Magic
Expose your Gemini API Key to enable live, dynamic executive board debates and copywriting:
```bash
export GOOGLE_API_KEY="your_gemini_api_key"
```

### Step 3: Run Interactive Simulations
Now, run any of the terminal-guided cyber-commands:

*   **⚡ Show Boardroom AI Debate & Voting**:
    ```bash
    python cli.py run --objective "Build a self-driving electric scooter"
    ```
*   **⚡ Run AI Git PR Writer on current workspace changes**:
    ```bash
    python cli.py git
    ```
*   **⚡ Generate Instagram, LinkedIn, and Twitter launches with mockup assets**:
    ```bash
    python cli.py marketing --name "AI Cup" --desc "A self-warming mug that analyzes your mood"
    ```
*   **⚡ Run API Server & Interactive Swagger Docs**:
    ```bash
    python cli.py serve
    ```
    Once started, navigate to `http://localhost:8000/api/docs` to interact with our advanced endpoints!
